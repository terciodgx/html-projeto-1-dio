<!DOCTYPE html> – Define que o documento é do tipo HTML5.

<html> – Tag raiz de um documento HTML.

<head> – Contém metadados do site, como título e codificação.

<meta charset="UTF-8"> – Define a codificação de caracteres, permitindo acentuação.

<meta name="viewport"> – Faz o site se adaptar a diferentes tamanhos de tela (responsividade).

<title> – Define o título da aba do navegador.

<body> – Onde todo o conteúdo visível pelo usuário é colocado.

<i> – Deixa o texto em itálico.

<strong> – Destaca o texto em negrito (semântico).

<u> – Sublinha o texto.

<br> – Cria uma quebra de linha.

<p> – Define um parágrafo de texto.

<h1> até <h6> – Criam títulos, sendo <h1> o maior e mais importante, e <h6> o menor.

<input> – Campo de entrada para o usuário digitar dados (texto, número, senha, etc).

<img> – Exibe uma imagem na página. Usa os atributos src (origem) e alt (descrição).

Atributo id – Identifica um único elemento na página, útil para JavaScript.

Atributo class – Define um grupo de elementos que podem ser estilizados juntos com CSS.

Atributo style – Aplica estilos diretamente no elemento (ex: cor, tamanho).

<sup> – Eleva um número ou símbolo (ex: potência).

<blockquote> – Citação longa de outra fonte, com recuo no texto.

<q> – Citação curta dentro de uma linha de texto.

<ol> – Lista ordenada (numerada).

<ul> – Lista não ordenada (com marcadores).

<li> – Itens de listas (<ul> ou <ol>).

<header> – Cabeçalho da página ou seção.

<nav> – Área de navegação, onde ficam os links principais.

<a> – Cria um link clicável (âncora).

<main> – Conteúdo principal da página.

<section> – Seção genérica de conteúdo.

<aside> – Conteúdo lateral, como anúncios ou links extras.

<footer> – Rodapé da página.

<article> – Bloco de conteúdo independente, como postagens ou notícias.

<time> – Representa uma data ou horário.

<audio> – Reproduz arquivos de áudio na página.

<video> – Reproduz arquivos de vídeo.

<source> – Usado dentro de <audio> ou <video> para indicar arquivos de mídia alternativos.

<track> – Adiciona legendas ou texto para vídeos.

<form> – Cria um formulário para coleta de dados do usuário.

<label> – Define uma etiqueta para os campos do formulário.

<button> – Botão que pode enviar o formulário, limpar, ou executar ações JavaScript.

<select> – Menu suspenso com várias opções.

<option> – Cada item dentro do menu suspenso (<select>).

<textarea> – Campo de texto longo, ideal para mensagens ou observações.

<fieldset> – Agrupa elementos de um formulário visualmente.

<legend> – Legenda para o grupo de campos definidos por um <fieldset>.

<table> – Cria uma tabela para exibir dados organizados em linhas e colunas.

<caption> – Título da tabela.

<thead> – Cabeçalho da tabela.

<tbody> – Corpo principal da tabela (onde ficam os dados).

<tfoot> – Rodapé da tabela (ex: totais).

<tr> – Linha da tabela.

<th> – Cabeçalho de uma coluna da tabela.

<td> – Célula de dados da tabela.

<div> – Container genérico em bloco. Muito usado para estruturar o layout.

<span> – Container genérico em linha. Ideal para estilizar partes específicas do texto.

<figure> – Container para imagens com legenda.

<figcaption> – Legenda de uma imagem dentro de uma <figure>.

<picture> – Carrega diferentes imagens dependendo do tamanho da tela (responsivo).

<iframe> – Incorporar conteúdo externo como vídeos, mapas, etc. Deve-se ter cuidado com segurança.
